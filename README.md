# Generador de Contraseñas en Python:

Este proyecto es un **generador de contraseñas seguras** desarrollado en Python.

# Características:
- Genera contraseñas aleatorias y seguras.
- Permite configurar:
  - Longitud de la contraseña.
  - Inclusión de letras mayúsculas y minúsculas.
  - Números.
  - Caracteres especiales.
- Código simple y fácil de entender.
- Ideal para practicar lógica de programación y seguridad básica.

# Requisitos:
- Python 3.8 o superior.
- No requiere librerías externas (usa módulos estándar como `random` y `string`).

# Uso:
1. Clona o descarga el repositorio en tu computadora.
2. Abre una terminal en la carpeta del proyecto.
3. Ejecuta el archivo principal:
   ```bash
   python generador.py
